# Diagram maps

Hand-drawn architecture maps for this role, stored alongside the [graphify](https://github.com/safishamsi/graphify) vault. Generated locally — not committed to GitHub.

## How to add a map

1. Bootstrap vault: `./scripts/setup-knowledge-vault.sh` (full corpus: `/graphify .`)
2. In Cursor, ask to map something (e.g. "diagram the CI pipeline in Obsidian")
3. Agent uses **obsidian-diagrams** + **drawio-skill** skills
4. Open this vault in Obsidian: `graphify-out/obsidian/`
